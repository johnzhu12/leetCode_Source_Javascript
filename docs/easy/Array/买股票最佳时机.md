
# [买股票最佳时期](https://leetcode-cn.com/problems/best-time-to-buy-and-sell-stock-ii/)

![maxProfit](./imgs/maxProfit.png)

```js
/**
 * @param {number[]} prices
 * @return {number}
 */
var maxProfit = function(prices) {
  let money = 0
  let buy = prices[0]
  for (let price of prices) {
    if (price > buy) {
      money += price - buy
    }
    buy = price
  }
  return money
};
```
